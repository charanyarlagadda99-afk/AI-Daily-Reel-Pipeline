# AI-Daily-Reel-Pipeline
An autonomous agentic workflow built in n8n that transforms daily AI news into ready-to-post social media reels.  Key Features:  Content Sourcing: Automatically fetches the latest AI news via RSS.  Agentic Scripting: Uses Groq-powered LLMs to summarize news and generate viral-style scripts.  Voice &amp; Visuals: Integrates Murf TTS for high-quality .
